# site-compare

Обход только HTML, сравнение H1/Title/Description, учёт различий по завершающему слешу, визуальный просмотр.
Запуск:
1) npm i
2) cp config.example.json config.json (укажите домены)
3) npm run crawl
4) npm start
(опц.) npm run snap
